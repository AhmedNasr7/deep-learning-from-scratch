# Contrastive Learning

This module is a from-scratch implementation scaffold for contrastive learning.
It covers both classic image contrastive learning and masked transformer-style
self-supervised pretraining.

Planned components:
- `simclr.py` — SimCLR style contrastive representation learning
- `mae.py` — Masked Autoencoder / masked transformer pretraining
- `losses.py` — contrastive and reconstruction losses
- `train.py` — training script scaffold
